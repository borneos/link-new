This is a borneos-link apps Fullstack with NextJs with route pages | Prismaio | Mysql | NextAuth | ShadcnUI

## Getting Started

First, install all package:

```bash
npm run install
# or
yarn install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
```
